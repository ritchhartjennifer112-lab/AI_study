**作者**: IE负责人

## 1. 背景与目标

### 1.1 项目背景

- 工厂产品：铜牌仓设备、冲剪机、折弯机、单头铣、双头铣、打标机、20工位抽屉线等
- 生产模式：ETO（Engineer to Order），每单规格不同但结构同族
- 约束：单人执行、两周交付、无系统支持、车间极度繁忙无配合
- 现有数据：7种设备的历史实际工时（钳工+电工），1份冲剪机SOP（33工步）
- 新增约束：**大件搬运频繁，实际很重，需考虑吊装系数与疲劳宽放**

### 1.2 核心目标

交付一套**参数化标准工时Excel计算器**，实现：

1. 输入设备关键参数 → 自动估算标准工时
2. 同产品族内，不同规格设备可复用公式
3. 历史实际工时与标准估算对比校验
4. 数据质量分级标记（A/B/C/D/E）

## 2. 产品族划分

| 族 | 设备 | 核心结构特征 | 覆盖SOP |
|---|---|---|---|
| **族A：单机加工设备** | 冲剪机、折弯机、单头铣、双头铣、铜排加工设备 | 床身→立柱/机架→导轨→丝杆传动→夹钳→电机座→调试 | 冲剪机SOP（33步）覆盖约80% |
| **族B：龙门/仓储设备** | 铜排仓、尾料仓 | 龙门立柱→横梁→升降机构→料架→抓手/移栽→仓冲连线 | 无SOP，依赖历史数据+IE估算 |
| **族C：生产线设备** | 20工位抽屉线、打标机产线 | 型材框架→多工位站点→输送轨道/链板线→联动调试 | 无SOP，依赖历史数据+IE估算 |

**划分依据**：
- 族A设备共享"机床式"结构（落地→机架→导轨→传动→执行→调试）
- 族B设备共享"门架式"结构（立柱→横梁→升降→料架→搬运）
- 族C设备共享"分布式产线"结构（框架→站点→轨道→联动）

| A-04 | 支架/板件固定 | 支架数量、每支架螺丝数、螺丝规格、可达性 | 垫板(24颗)、V型盒、固定块、夹钳支架、拖链底板、滚筒支杆 |

| A-08 | 整机调试 | 组件总数、功能联调复杂度 | 丝杆滑动检查、夹钳动作确认、传感器信号测试 |

## 4. 电工工时参数化设计

### 4.1 电工工时结构

每台设备的电工工时 = 电柜接线工时 + 外围线布线工时

### 4.2 电柜接线工时公式

```
电柜接线工时 = 端子总数 × 单位端子接线时间
             + 断路器/开关数量 × 单件安装基准
             + 继电器/接触器数量 × 单件安装基准
             + 指示灯/按钮数量 × 单件安装基准
             + PLC/控制器数量 × 单件安装基准
             + 控制柜大小系数 × 柜内布线附加时间
```

**参数化驱动因素**（需明天向于立阳/老宋确认的数据）：

| 参数 | 获取来源 | 说明 |
|------|---------|------|
| 端子总数 | 设计BOM/接线图 | 二次回路端子数量 |
| 断路器数量 | 元件清单 | 主开关、分路开关 |
| 继电器数量 | 元件清单 | 中间继电器、时间继电器 |
| 指示灯/按钮数 | 元件清单 | 操作面板元件 |
| 控制柜规格 | 设计图纸 | 柜体尺寸（宽×高×深） |
| PLC数量 | 元件清单 | 有无PLC及I/O点数 |

### 4.3 外围线布线工时公式

```
外围线工时 = 电机功率线布线长度(m) × 单位长度布线基准 × 线径系数
          + 传感器/开关线缆数量 × 单根布线基准
          + 动力柜到设备主线缆长度(m) × 单位长度基准
          + 线槽安装长度(m) × 单位长度线槽安装基准
          + 接线端子压接数量 × 单只压接基准
```

**参数化驱动因素**（需明天确认）：

| 参数 | 获取来源 | 说明 |
|------|---------|------|
| 电机数量及功率 | 设备参数表 | 决定动力线规格和长度 |
| 传感器/开关数量 | 设备参数表 | 决定信号线数量 |
| 设备到配电柜距离 | 现场布局 | 决定主线缆长度 |
| 线槽走向长度 | 现场布局 | 决定线槽安装量 |
| 线缆穿管/桥架 | 现场条件 | 影响可达性系数 |

### 4.4 电工参数化数据需求清单

**请于立阳/老宋提供以下信息：**

1. **典型设备的元件清单**（1~2种即可）
   - 控制柜内有哪些元件？各多少个？
   - 控制柜尺寸多大？

2. **典型设备的电机配置**
   - 几个电机？功率多大？
   - 电机到控制柜的布线距离大概多少米？

3. **典型设备的传感器配置**
   - 几个光电开关？几个接近开关？
   - 传感器到控制柜的线缆长度？

4. **线槽/桥架的走向**
   - 是走地面线槽还是空中桥架？
   - 线缆是否需要穿管？

## 5. 核心修正系数体系

### 5.1 空间可达性系数

| 场景 | 系数 | 说明 |
|------|------|------|
| 敞开操作（正面站立） | 1.0 | 打标机、折弯机 |
| 半受限（需侧身或弯腰） | 1.1~1.2 | 冲剪机底部、抽屉线内侧 |
| 受限（需蹲下或仰面） | 1.2~1.4 | 铜排仓底部横梁 |
| 盲操作/柜内操作 | 1.4~1.6 | 控制柜内部接线 |
| 登高操作（需爬梯或平台） | 1.5~2.0 | 铜排仓顶部、抽屉线高处 |

### 5.2 吊装/辅助系数（新增，基于用户现场观察）

| 场景 | 系数 | 说明 |
|------|------|------|
| 单人可搬动（<20kg） | 1.0 | 标准件、小钣金 |
| 双人搬运（20~50kg） | 1.2 | 中型支架、电机 |
| 需行吊/叉车（50~200kg） | 1.4~1.6 | 大型床身、龙门立柱 |
| 需多人+行吊（>200kg） | 1.6~2.0 | 整机机架、大型料架 |
| **大件频繁搬运疲劳附加** | **+0.1~0.3** | 同一工步中连续搬运3次以上 |

### 5.3 对准精度系数

| 精度要求 | 系数 | 说明 |
|----------|------|------|
| 目视对准 | 1.0 | 支架粗定位 |
| 螺孔对齐 | 1.1~1.2 | 一般固定 |
| 导轨平行度/直线度 | 1.2~1.4 | 直线导轨安装 |
| 丝杆同心度/回丝滑入 | 1.3~1.5 | 丝杆精密装配 |
| 夹钳对中精度 | 1.2~1.4 | 夹钳机构 |

### 5.4 重复操作衰减系数

```
第1颗螺丝 = 1.0
第2~5颗 = 0.95
第6~10颗 = 0.90
第11~20颗 = 0.85
第21颗以后 = 0.80
```

适用于同规格、同位置、连续拧紧的螺丝（如导轨安装48颗M6×25）。

### 5.5 设备型号基础系数

| 设备规模 | 系数 | 典型设备 |
|----------|------|----------|
| 小型 | 0.7~0.8 | 打标机 |
| 中小型 | 0.9~1.0 | 折弯机、单头铣 |
| 中型 | 1.0~1.1 | 双头铣、冲剪机 |
| 中大型 | 1.2~1.3 | 铜排仓 |
| 大型 | 1.4~1.6 | 20工位抽屉线 |

## 6. Excel计算器结构

### 6.1 Sheet 1：装配单元基准时间库

| 字段 | 说明 |

| 钳工标准工时(H) | 自动计算 |
| 电工标准工时(H) | 自动计算（电柜+外围线） |
| 总标准工时(H) | 钳工+电工 |
| 数据来源标记 | 各单元数据来源汇总 |

### 6.3 Sheet 3：历史数据校验表

| 字段 | 说明 |
|------|------|
| 设备型号 | |
| 历史实际钳工工时(H) | 从现有Excel汇总 |
| 历史实际电工工时(H) | 从现有Excel汇总 |
| 参数化估算钳工工时(H) | Sheet 2结果 |

| 偏差率 | (估算-实际)/实际 |

## 8. 风险与退路

| 风险 | 触发条件 | 退路 |
|------|---------|------|
| 历史数据无法拆解反推 | 冲剪机323H拆到单元后，各单元时间明显不合理 | 退回方案A：只做"设备-历史平均工时"汇总表 |
| 跨设备套用偏差过大 | 折弯机公式估算60H，实际43H，偏差>40% | 把族A拆为"小型机"和"中大型机"两个子族 |
| 电工参数化数据无法获取 | 于立阳无法提供元件清单和布线参数 | 电工部分退化为"历史平均值"，只参数化钳工 |
| 大件搬运系数不准 | 反推的吊装时间与实际观察差距大 | 加入"现场实测"项，用用户实际观察数据校准 |
| 两周做不完 | D10还没完成Sheet 1填充 | 优先完成族A（单机设备），族B/C用简化版 |

## 9. 新增约束记录

### 9.1 大件搬运系数（2026-05-18 用户现场观察补充）

用户实际参与搬运后反馈：机台加工存在大量大件，**实际很重**，需要在系数和宽放中特别考虑：

- **吊装系数**：大件（>50kg）必须使用行吊，吊装辅助时间不可忽略
- **疲劳宽放**：连续搬运大件需增加疲劳恢复时间（建议+10%~15%）
- **多人协同系数**：大件需多人配合，协同效率低于单人操作
- **安全宽放**：大件搬运的安全检查时间（吊具确认、路径确认）

**设计调整**：在A-01（落地调平）和B-01（立柱安装）等涉及大件的单元中，强制加入"单件重量"参数，重量>阈值时自动触发吊装系数和疲劳宽放。
